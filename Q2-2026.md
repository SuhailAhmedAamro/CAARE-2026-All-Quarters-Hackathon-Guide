## ðŸ”¥ Q2 Hackathon: "The Agentic Challenge"

### Goal
- 48-Hour Hackathon jisme **Autonomous Agentic System** banana hai jo bina insani madad ke complex tasks complete kare.

### Example Tasks
- **Auto-Researcher Agent**: Internet se data collect karke summary banaye aur Docusaurus par naya page publish kare.
- **Code-Reviewer Bot**: GitHub code check kare, ghalti find kare aur PR bheje.
- **Physical AI Simulator Agent**: NVIDIA Isaac Sim mein robot ko instructions de kar maze solve karwaye.

### Grading Criteria
- **Agent Autonomy**: AI ne kitne faisle khud liye? (30%)
- **Tool Usage**: Kya AI ne sahi waqt par sahi tool use kiya? (30%)
- **Spec-Compliance**: Project Governess ke rules follow kar raha hai? (20%)
- **UI/UX**: Dashboard professional hai? (20%)
